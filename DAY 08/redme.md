# 📇 Contact Manager

**Contact Manager** is a backend application built in **Java** to manage personal or business contacts using full CRUD operations.

## ✨ Features

- ➕ Add new contacts
- 📖 View contact list or single contact
- 🛠️ Update contact information
- ❌ Delete contacts

## 🛠️ Tech Stack

- React
- Express.js
- MongoDB
